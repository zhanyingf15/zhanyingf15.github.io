使用maven搭建maven仓库，找了网上的一些方法，踩了写坑，鼓捣了大半天终于弄好了，特此记录备忘。
网上基本上有两种方法，但本质上都是一样的，一种是将工程编译到本地，再将本地编译好的jar等文件上传到GitHub。另一种是在pom文件中配置插件，将编译上传使用插件完成。
